Overview

  • JavaScript (JS) can be used to create, manipulate, and remove HTML elements
  • Most of the code you write will be JS
  • JS inspired by functional programming
    • Supports logical operators(if, else, etc.)
    • Many mathematical operators (+, -, *, /) are similar

JavaScript Variables

There are various data types used in Javascript

  • Numeric: a single quantitative value
    • var x = 13;
  • String: a string of characters, defined using quotation marks
    • var x = "hello";
  • Array: A set of items (string or numeric) stored in a vector
    • var x = [13, 14, 15];
    var x = ["string one", "string two", "string three"];
  • Object: An item which holds key-value pairs
    • var x = {name:'mike'};

Numbers and Strings

  • There is only one type of JavaScript number
  • Mathematical operators can be used, as well as advanced Math functions
    • var x = 2*15 //x is 30
    var x = Math.sqrt(4) //x is 2
  • JS supports inline concatinations and common string functions and
    • var name = "Mike"
    var sentence = "Hello, my name is " + name //Hello, my name is Mike
    var shortened = sentence.replace("Hello, ", "") //my name is Mike

Arrays

  • Arrays are vectors of comma seperated items, denoted by square brackets []
    • var x = [13, 14, 15]
  • Values of arrays are accessed by their index, starting at 0
    • x[0] //13

Objects

  • Objects are data items (often key-value pairs), that have properties and methods
    • var person = {firstName:'Anna', lastName:'Smith', height:"5'4", age:32}
  • Access properties from objects by referencing the key
    • person.firstName // Anna
    person['firstName'] // Anna

Functions

  • Functions perform blocks of code on parameters that are passed to them
  • Any function returns a single item (object, array, value, etc)
    • var myFunction = function(a,b) {return a*b})
  • Functions are executed as follows
    • var result = function(2,4) //result is 8

Manipulating HTML Elements

Creating all page elements with HTML would be tedious (impossible)

  • JavaScript can be used to change page element content and style
    • document.getElementById("demo").innerHTML = "New Text!"
    document.getElementById("demo").style.fontSize = "25px";
  • D3 extends this type of manipulation to multiple elements
