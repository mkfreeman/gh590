Overview

Javascript(JS)

  • The most popular programming language in the world
  • Most of the code you write will be JS
  • Can be used to create, manipulate, and remove HTML elements
  • Program behaviors of the webpage
  • Handle data
  • Written in a separate file, referenced by index.html
    • <script type="text/javascript" src="js/myScript.js"></script>

Javascript is a scripting language

  • Differs substantially from markup languages
  • Creates variables which are not represented in the DOM
  • Supports structured programming (ie, conditional if, else) statements
  • We will aim for a functional understanding of JS

JavaScript Variables

There are multiple variable types used in JavaScript

  • Numeric: a single quantitative value
    • var x = 13;
  • String: a string of characters, defined using quotation marks
    • var x = "hello";
  • Array: A set of items (string or numeric) stored in a vector
    • var x = [13, 14, 15];
    var x = ["string one", "string two", "string three"];
  • Object: An item which holds key-value pairs
    • var x = {name:'mike'};
  • Function: An executable block of code that returns a value
    • var timesTwo = function(a){return a*2};

JS variables have different properties and methods based on type

  • Properties describe a feature of the element
    • 
var characters = "This is a string"
characters.length // Returns the length of the variable
var arr = [1,2,3]
arr.length // Returns the number of items in the array
  • Methods are actions that variables can perform
    • 
var characters = "This is a string"
characters.replace("This", "Here") // Replaces "This" with "Here"
var arr = [1,2,3]
arr.push(4)// Pushes the number 4 into the array as the last element

Numbers and Strings

Numbers

  • There is only one type of JavaScript number
  • Mathematical operators can be used, as well as advanced Math functions
    • var x = 2*15 //x is 30
    var x = Math.sqrt(4) //x is 2

Strings

  • String variables are assigned by wrapping characters in single or double quotes
    • var x = 'using single quotes'
    var x = "or double quotes, if you's have a single quote in the string"
  • JS supports inline concatenations and common string functions
    • var name = "Mike"
    var sentence = "Hello, my name is " + name //Hello, my name is Mike
    var alternate = sentence.replace("Hello", "Yo") //Yo, my name is Mike

Arrays

  • Arrays are vectors of comma seperated items, denoted by square brackets []
    • var x = [13, 14, 15]
  • Values of arrays are accessed by their index, starting at 0
    • x[0] //13
    x[1] //14

Objects

  • Objects are data items with multiple key-value pairs
    • var person = {firstName:'Anna', lastName:'Smith', height:"5'4", age:32}
  • Access values from objects by referencing the key
    • person.firstName // Anna
    person['firstName'] // Anna

Functions

  • Functions perform blocks of code on parameters that are passed to them
  • Any function returns a single item (object, array, value, etc)
    • var myFunction = function(a,b) {return a*b})
  • Functions are executed as follows
    • var result = myFunction(2,4) //result is 8

Conditional Statements (flow control)

  • Execute statements only if a condition is true
    • 
	var a = 1;
var b = 2;
if(a<b) {
	//take some action
}
  • You can also provide a default alternative
    • 
	if(a<b) {
	//take some action
}
else {
	//take a different action
}
  • Or provide multiple conditions
    • 
	if(a<b) {
	// take some action
}
else if (a == b){
	// take some action
}
else {
	// take the default action
}

Switch case

  • An extension of if, else, but for multiple conditions
  • Benefit is you don't have to re-type conditions
    • 
	var country = 'USA'
switch(country) {
	case 'USA':
		// take some action
		break;
	case 'FRA':
		// take some action
		break;
	default:
		// take default action
		break;
}

Functions can take functions as parameters

Imagine you have a function that compares two values


var comparison = function(a,b) {
	if(a>b) {
		return 'a is greater that b'
	}
	else {
		return 'b is less than or equal to a'
	}
}

You might want to do something in one of those scenarios


if(a>b) {
	// do a specified action
	return 'a is greater that b'
}

You can specify the action by passing in a function


var comparison = function(a,b, callback) {
if(a>b) {
	callback()
	return 'a is greater that b'
}
else {
	return 'b is less than or equal to a'
}
}

Advanced array manipulation

Execute a function on each element of an array by using the "map" method

  • The map method is a built in alternative to writing loops
  • Each array value is passed to a provided function
  • Values in the array are unchanged

	var arr = [1,2,3]
var timesTwo = function(d) {return d*2}
var doubleArr = arr.map(timeTwo) // returns [2,4,6]

Map functions can also be written in-line


	var arr = [1,2,3]
var doubleArr = arr.map(function(d) {
	return d*2
})

You can filter arrays using a similar syntax

  • Filtering only returns values that meet a given conditionality
    • 
	var arr = [1,2,3]
var greaterThanTwo = arr.filter(function(d) {
	return d>2
})

    Sorting works in a similar fashion, but:

    • Sorting functions will change your array (unlike map, filter)
    • Sorting functions take two arguments
    • These parameters represent two values to be compared
      • 
var arrToSort = [1,4,3,5,2]
sortFunction = function(a,b) {
	return a - b
}
arrToSort.sort(sortFunction)
