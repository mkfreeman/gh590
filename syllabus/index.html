Course Description

The purpose of this course is to provide students with the conceptual and technical skills necessary to design and build interactive data visualizations using global health data.

Conceptually, the course will focus on skills necessary for designing clear, communicative, and powerful visualizations, specifically:

  • Understanding the role of interactive data visualization in global health
  • Processes for designing visualizations for a targeted audience
  • Best practices for encoding variables with graphical properties
  • Novel approaches for communicating global health messages using visualizations

Technically, the course will cover the skills necessary to build interactive visualizations on the web, including:

  • How to set up and maintain webpages
  • A fundamental understanding of webpage properties (HTML, CSS)
  • The ability to process and manipulate data for the web using Javascript
  • Building an array of interactive charts using the Data Driven Documents (D3) library

Students will be evaluated based on their ability to complete tasks that combine both technical and conceptual skills (i.e., not just build a visualization, but build a visualization that properly uses graphical encodings to communicate a directed message to a specified global health audience).

Course structure will follow a mixed lesson/lab format, with at least 50% of each day devoted to hands-on programming experience by students. Bring your laptop or arrange with the instructor to have computer resources available for you in class.

Syllabus

Week 1, Day 1

The role of interactive data visualization in global health (link)

To begin the course, we will explore the ways in which interactive data visualization has transformed the global health space. Descriptive epidemiology, impact evaluation, and disease prioritization have all been impacted by the use of (interactive) visualizations.

What is a website? Introduction to HTML and CSS (link)

Introduction to the properties and styles of webpages. Students will use online platforms to experiment with simple HTML elements and adjusting their styles via CSS.

Readings

Information Visualization: Using Vision to think. Card, Mackinlay, and Schniderman

 

Week 1, Day 2

User driven design for global health (link)

The user driven design process that we will focus on challenges visualization developers to understand the goals, questions, and skill levels of potential users. Understanding the users in the global health field is fundamental to developing visualizations that can be effective and impactful.

Introduction to Javascript (link)

Javascript, the most popular programming language in the world, is a dynamic scripting language that be used to manipulate HTML elements. We will cover javascript object properties, and practice using common functionality.

 Readings 

Adapting User-Centered Design Methods to Design for Diverse Populations.  Putnam et. al.  (link)

User-Centered Design. Wikipedia. (link)

 

Week 1, Day 3

Graphical perception (1/2)

Understanding how humans visually perceive graphical properties is the foundation of building visualizations. This first section covers a typology of datatypes (nominal, ordinal, categorical) and pre-attentive cognition.

D3: Data Driven Documents (1/2) (link)

This lectured introduces the Javascript library D3. D3, the most commonly used web- visualization package, enables users to dynamically create, transition, and manipulate HTML elements.

Readings

Graphical Perception: Theory, Experimentation, and Application to the Development of Graphical Methods. Cleveland and McGill

 

Week 1, Day 4

Graphical perception (2/2)

Continuing the theme of graphical perception, this lecture discusses a hierarchy of effectiveness for graphical encodings based on different datatypes. We will cover examples within health that have successfully (and not successfully) used these principles.

D3: Data Driven Documents (2/2) (link)

An extension of the previous D3 lecture – adding, removing, and manipulating different types of elements on a webpage.

Readings

D3: Data Driven Documents. Bostock, Ogievetsky, and Heer

 

Week 1, Day 5

Interactivity

This lecture examines effective ways to leverage interactivity in data visualizations. We will consider visual queues that imply interactive methods, and consider how effective these queues are for different users in global health.

Technical review

The first week’s skills will be reviewed and practiced with exercises that combine elements across the entire week.

Readings 

The Eyes Have it: A Task by Data Type Taxonomy for Information Visualization. Ben Shneiderman

 

Week 2, Day 1

Everyone loves maps

Global health information is frequently displayed on maps, but in which situations are maps effective visualizations? This lecture explores examples of maps as data visualizations, and challenges when they are effective.

Everyone loves maps

The D3 package has built in functionality for building maps and coloring each location based on a dataset. Today, we’ll build a map using D3.

Readings

How to Use Maps in Data Visualization. Drew Skau (link)

Week 2, Day 2

Graphic design

The aesthetic components of a website largely determine how a user interprets the quality of information, but what is it about a visualization that makes it look good, dated, cool, scientific, or anything else? We’ll discuss elements of graphic design that help frame a visualization in an appropriate and aesthetically pleasing way.

Buttons, dropdowns, and sliders

To build selection functionality for visualizations, we’ll use the Javascript library jQuery to create multiple types of controls which can be used to update the data being shown in a visualization.

Readings

Practical Rules for Using Color in Charts. Steven Few (link)

 

Week 2, Day 3

Visualization ethics and evaluation

Given our goal of impacting global health, what are the ethical implications involved with visualizations?  We will also consider how to conceptualize and measure the effectiveness of a visualization.

D3 layouts

In this exercise, we’ll explore D3’s built in algorithms for organizing data into different spatial layouts, including proportional and hierarchical data (stacked bar charts, treemaps, etc.)

Readings

Crowdsourcing Graphical Perception: Using Mechanical Turk to Assess Visualization Design. Heer and Bostock (link)

 

Week 2, Day 4

Narrative visualizations in global health

In large health datasets, there are myriad stories to be told, so how can you build specific, compelling stories into an interactive data visualization? We will examine examples, and provide a framework for integrating directed stories into visual tools.

Building Applications

Designing more advanced applications such as multiple integrated charts, or multiple views of the data requires more generalizable methods of programming —  this lecture introduces these approaches.

Readings

Narrative Visualization: Telling Stories with Data. Segel and Heer (link)

 

Week 2, Day 5

Project Presentations

On the final day, students will share their visualization tool, including both a description of the conceptual process as well as a demonstration of the interactive tool.  Presentations are scheduled to happen at the Institute for Health Metrics and Evaluation - guests are welcome.
