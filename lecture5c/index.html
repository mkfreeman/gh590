Tasks and actions

What does a user want to accomplish through their interactions (tasks)?

  • Select view of data (chart type)
  • Filter data
  • Retrieve information
  • Zoom into data
  • Highlight data
  • Rearrange data (sort)
  • Explore specific sections, relationships

What steps does a user take to accomplish their tasks (actions)?

  • Hover
  • Click
  • Click + drag (to move)
  • Click + drag (to select: brush)
  • Mousewheel up/down
  • Type

User versus developer driven interaction

  • User driven: user seeks out information of interest
  • Developer driven: visualization exposes information to users in specific steps
