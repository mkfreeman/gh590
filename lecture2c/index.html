UCD Overview

  • Prioritizes end users at each stage of the design/development process
  • Aims to meet needs/wants of users(not change behavior)
  • Engages users for design, prototyping, testing (ideally)
  • Provides a framework for prioritization and evaluation

UCD Process

  • Identify target population for product (users)
  • Construct aggregate representations of common users (personas)
  • Describe contexts in which product is used (scenarios)
  • Detail specific tasks to accomplish (use cases)

Users/Personas

  • Users are identified through research / estimation
  • Personas are developed to represent groups of users
  • Development is centered around the personas
  • Crucial for understanding the needs, wants, and limitations of users
  • Often have multiple (competing) personas

Persona Description

Typically include:

  • Name
  • Demographics (age, language)
  • Profession
  • Roles and responsibilities
  • Goals and tasks
  • Work environment

Binding data

  • The power of D3 is its ability to bind data to DOM elements
  • Each element (ie, <circle>,<rect>) is tied to data
  • Any data without existing HTML elements can create new HTML elements (enter)
  • Any HTML elements without corresponding data can be removed (exit)
  • Existing HTML elements can transition to reflect updates in the data (update)

Enter Elements

  1. Select element to which you want to bind data (ie, svg)
    2. var mySvg = d3.select('#my-svg')
  2. Select all child elements of SVG that you want to bind data to
    3. var circles = mySvg.selectAll('circle').data(dataSet)
  3. Enter new objects onto the page and append the desired elements
    4. circles.enter().append('circle')
  4. Assign attributes
    5. circles.attr('r', 15).attr('cx', 50)...

Remember functions?

  • Many similar processes are repeated when updating elements
  • These processes can (should) be stored in functions
  • Function to define circle attributes
    • 
var circleFunction = function(circle) {
	circle.attr('r', 15)
	  .attr('cx', function(d) {return d*1})
	  .attr('cy', function(d) {return d*30})
	  .attr('fill', 'blue')
}

Exit Elements

  1. Select element to which you want to manage data (ie, svg)
    2. var mySvg = d3.select('#my-svg')
  2. Select all child elements of SVG that you want to bind data to
    3. var circles = mySvg.selectAll('circle').data(dataSet)
  3. Remove circles that are no longer present
    4. circles.exit().remove()

Transition Elements

  1. Select element to which you want to manage data (ie, svg)
    2. var mySvg = d3.select('#my-svg')
  2. Select all child elements of SVG that you want to bind data to
    3. var circles = mySvg.selectAll('circle').data(dataSet)
  3. Select all elements and transition a property
    4. mySvg.selectAll('circle').transition().duration(500).attr('cx', 100)
