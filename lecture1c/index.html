Course goals

Provide students with the conceptual and technical skills necessary to design and build interactive data visualizations

Conceptual skills

  • Develop a process for answering specific questions with visualizations
  • Identify best graphical layouts for different data types
  • Best practices for encoding variables with graphical properties

Technical skills

  • Build interactive visualizations using D3.js
  • Publish visualizations on the web
  • Independently solve technical challenges encountered

Course structure

Week 1

  • Conceptual: Foundational theories of design approaches and graphical perception
  • Technical: Foundational understanding of web programming and D3
  • Assignments: Specific assessments of skills

Week 2

  • Conceptual: Various topics in visualization (maps, design, evaluation)
  • Technical: Tutorials on using specific chart types (treemaps, stacked bars, etc.)
  • Assignments: Independent project work

Daily process

  • Conceptual lecture (20-30 mins)
  • Conceptual exercise (20-30 mins)
  • Technical lecture (1 hour)
  • Technical exercises (1-2 hours)
  • Exercise review(30-60 mins)

Evaluation

  • Class participation: 10%
  • Homework assignments: 35%
  • Final project: 55%
