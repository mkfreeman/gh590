Overview

The internet (briefly)

Browsers

  • Chrome, firefox, IE
  • Retrieve and present information they are directed to via URLs
  • Information (files) can be stored on
    • Your local machine (i.e., file:///)
    • A remote web server (i.e., HTTP://)
    • A local web server (i.e., localhost:8888/)
  • The protocol for handeling the information is dictated by the URL prefix (ie, HTTP, file:///, etc.)

Web pages

  • Browsers render files into page elements
    • <text style="color:red">RED TEXT</text>
    RED TEXT
  • Page elements (text, images, etc.) are assembled into a tree structure calle the Document Object Module (DOM)
  • These elements can be inspected using the browser (rt. click)

Elements of a webpage

HTML

  • HyperText Markup Language
  • Uses <tags> to instruct the browser how to represent elements
  • Elements begin with an opening <tag>, have some content, and close with a closing </tag>

Common HTML Elements

  • <div>: Division of a page, used to group elements
  • <text>: A piece of text on the page
  • <ul>: An unorder list (order list is <ol>)
  • <li>: A list element (such as this one)
  • <svg>:Scalable Vector Graphic, used as a container for graphical elements
  • <circle>:A circle element, such as a point on a scatterplot
  • <rect>:A rectangle element, such as a bar on a bar chart

All you have to do is learn how to arrange these shape and text elements on a page

HTML Element Attributes

  • HTML elements can be assigned attributes to define representation and behavior
  • These attributes can be assigned within the <tag>
  • Elements can be assigned a class for grouping and styling 
    Hello
  • Elements can also be assigned a unique ID for reference and styling 
    Title

Styles of a webpage

CSS

  • Cascading Style Sheets
  • Styles dictate how HTML elements are displayed
  • Styles can be assigned to:
    • A type of element (i.e., all <circle> elements)
    • A class (i.e., all <circle class="big"> elements)
    • An individual element by id (i.e., the one <circle id="my-circle"> element)
    • Any combination of these (i.e., circles with certain class)
  • Styles are inherited from parent objects (hence, cascading)
  • CSS should be written in a separate file, referenced by your index.html file

Syntax

  • CSS selects by ID using "#", by class using ".", or by listing the type of element (i.e., circle)
  • Style attributes are defined within {} for a selection, and each and with a semicolon
    • 
circle{
  color:blue;
}

.circle-class{
  stroke:2px;
}

#my-circle{
  visibility:hidden;
}
